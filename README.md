# Butterfly
A text-based TBBS with fantasy creatures, projectile weaponry, poisonous elixirs, and more.

```
Butterfly v0.16_04 | KILLS: 0   | DEATHS: 0   | SPARES: 0

PLAYR: Aradi  | HP: 26 | STR: 14

ENEMY: Goblin | HP: 2  | STR: 7

[FIGHT]
[ITEMS] <
[SPARE]
[EXIT ]

[1x POTION]
[0x SPEAR ] <
[1x POISON]
[BACK     ]
```

# Controls

```W, I, UP``` - up

```S, K, DOWN``` - down

```ENTER``` - select

```CTRL-C``` - close game

```Q``` - go back
